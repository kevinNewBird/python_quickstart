# 3.4.基础类型的转换
在python中，表示空的东西都是False，表示非空的东西都是True。</br>
比如：
-字符串。""是False，"ss"是True。</br>
-数字。0是False，非0是True。</br>
-数组。空数组为False，非空数组为True。
