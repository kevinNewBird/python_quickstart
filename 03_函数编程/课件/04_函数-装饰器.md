# 1.装饰器
装饰器本质上是一个闭包。</br>
作用：在不改变原函数调用的情况下，可以给函数前后添加新功能。（类似java中的代理）
## 1.1.语法
1.基础：
```python
def wrapper(fn): # wrapper: 装饰器，fn：目标函数
    def inner():
        # 在目标函数执行之前...
        fn()       # 执行目标函数
        # 在目标函数执行之后...
    return inner     # 千万别加()
```
